# 🐱🐶 Cat vs Dog Classifier (Deep Learning Mini Project)

## 📌 Project Overview

This project is a **Deep Learning-based Image Classification system** that predicts whether an input image is a **Cat 🐱 or Dog 🐶**.
The model is built using **TensorFlow/Keras** and trained on image data to achieve accurate classification.

---

## 🎯 Objectives

* Build a CNN model for image classification
* Understand preprocessing and data augmentation
* Train and evaluate a deep learning model
* Deploy a working classifier for real-world use

---

## 🧠 Technologies Used

* Python
* TensorFlow / Keras
* NumPy, Matplotlib
* Google Colab

---

## ⚙️ Project Workflow

1. Data Collection / Dataset Loading
2. Image Preprocessing (Resizing, Normalization)
3. Model Building (CNN Architecture)
4. Training the Model
5. Model Evaluation
6. Prediction on New Images

---

## 📊 Model Details

* Input Size: 224x224 images
* Model Type: Convolutional Neural Network (CNN)
* Loss Function: Binary Crossentropy
* Optimizer: Adam

---

## 🚀 How to Run

1. Open the notebook in Google Colab or Jupyter Notebook
2. Install required libraries (if needed)
3. Run all cells step by step
4. Upload an image to test prediction
